# C-Sharp-Latest-for-Unity
Unity editor utility that automatically sets C# version to latest available


|               | .NET 2.0 | .NET 4.6 | netstandard 2.0 |
|---------------|----------|----------|-----------------|
| Unity 2017.1+ |     6    |     6    |       N/A       |
| Unity 2018.1+ |    7.2   |  latest  |     latest      |
